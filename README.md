# nova_collection
整个攻防系统「pingsec」的核心模块，负责接收请求，完成具体的容器生成和下发等工作，以及端口映射到外网的工作。将成为本系统对企业合作，前后端分离，本项目仅负责后端部分，目前将鉴权、登录功能都集合在一起，后续会将这部分功能独立成一个项目即tokenstore
